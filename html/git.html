Git

  • git init Создать новый локальный репозиторий.
  • git clone Скопировать удалённый репозиторий на компьютер.
  • git add Подготовить файлы к сохранению (добавить в индекс)
  • git commit Сохранить подготовленные изменения локально.
  • git push Отправить локальные коммиты в удалённый репозиторий
